# Portfolio (Vite + React + Tailwind)

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```
3. Build:
   ```bash
   npm run build
   npm run preview
   ```

## Replace placeholders
- Replace [Your Name], short bio, and contact details in `src/components/*`.
- Add your resume at `src/assets/resume.pdf` and profile image `src/assets/profile-placeholder.png`.
- Update projects in `src/data/projects.js`.

## Deployment
- For Netlify/Vercel: connect repo and set build command `npm run build` and publish directory `dist`.
- For GitHub Pages: build and deploy the `dist` folder using an action or gh-pages.
